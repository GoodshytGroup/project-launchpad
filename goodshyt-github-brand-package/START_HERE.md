# GoodShyt Group GitHub Brand Package

Files included in this package folder:

- `README.md` — profile README template for the main developer account
- `ACCOUNT_SETUP.md` — profile, repo, and project setup guidance
- `REPO_DESCRIPTIONS.txt` — one-line repository descriptions
- `REPO_TOPICS.txt` — suggested GitHub topics for each repository
- `BANNER_CONCEPT.md` — brand and banner design rationale
- `ORG_PROFILE_README.md` — organization profile README template
- `REPO_COVER_PROMPTS.md` — prompts for generating matching repo cover images
- `ASSETS_NOTE.md` — note about the local PNG asset package
- `scripts/publish_profile_repo.sh` — local publish helper

Suggested use order:
1. Create the profile repo named exactly after the GitHub username.
2. Paste `README.md` into the root of that profile repo.
3. Apply `REPO_DESCRIPTIONS.txt` and `REPO_TOPICS.txt` to the target repositories.
4. Use `ORG_PROFILE_README.md` for the organization profile.
5. Generate or upload social preview images for each repository.
